# Observer — live market/learner checkpoints

`market-observer.ts` checkpoints roughly every 10 minutes while the app runs:
observations/min, symbols tracked, learner cycles + error, weights tracked.

- Newest ~48 stay here; the flood is archived to `archive/YYYY-MM/` (1,100+ notes).
- `archive/` is excluded from Obsidian search/graph (Settings → Files & links) — telemetry, not reading material.
- Machine-owned — never hand-edit.
