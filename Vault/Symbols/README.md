# Symbols — one hub page per watchlist ticker

Source of truth for membership: `AUTONOMOUS_WATCHLIST` in
`src/shared/constants.ts`. Every machine note wikilinks tickers as
`[[Symbols/<TICKER>]]`, so each page's **backlinks pane** accumulates that
symbol's full history by itself; each page also carries live `query` blocks
(mentions + trades) and a free-form operator-notes section.
